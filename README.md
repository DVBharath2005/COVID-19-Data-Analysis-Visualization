# COVID-19 Data Analysis & Visualization

A Python-based Data Analysis project that analyzes and visualizes COVID-19 data using powerful data science libraries such as Pandas, NumPy, Matplotlib, Seaborn, and Plotly.

---

# Project Overview

This project performs:

- Data Cleaning & Preprocessing
- COVID-19 Trend Analysis
- Country-wise Case Comparison
- Mortality Rate Analysis
- Vaccination Analysis
- Interactive Data Visualization
- World Map Visualization

The project helps in understanding the spread and impact of COVID-19 through graphical insights and statistical analysis.

---

# Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

```bash

```

